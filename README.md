# GitHub Dev Card Generator 🚀

A simple and modern web application that generates beautiful GitHub developer cards using GitHub profile data.

This project was built using FastAPI, Python, and frontend technologies to create visually appealing developer cards that can be shared in portfolios, resumes, and social profiles.

## 🌐 Live Demo

https://github-card-frontend-263881708620.us-central1.run.app/

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/debajoyatibera/github-dev-card-generator.git
```

Go to the project folder:

```bash
cd github-dev-card-generator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the server:

```bash
uvicorn main:app --reload
```

---

## 🚀 How It Works

1. Enter a GitHub username or profile
2. The application fetches GitHub data using the GitHub API
3. A developer card is generated automatically
4. Users can view, share, and download the generated card
